# Metrics

Quantitative tracking for learning effectiveness.

---

## Quality Scale

| Rating | Meaning | Criteria |
|--------|---------|----------|
| 5/5 | Excellent | Deep understanding, instant recall, can teach it |
| 4/5 | Good | Solid grasp, accurate recall with thought |
| 3/5 | Adequate | Basic understanding, partial recall |
| 2/5 | Weak | Fuzzy understanding, struggling recall |
| 1/5 | Failed | Cannot recall or fundamentally confused |
